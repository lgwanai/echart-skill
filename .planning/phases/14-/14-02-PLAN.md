  <action>
Create `scripts/server_status.py` with two main functions:

**1. get_chart_links(html_dir="outputs/html", port=None) function:**
- Scan `html_dir` for all `.html` files using `Path(html_dir).glob("*.html")`
- For each file, collect:
  ```python
  {
      "filename": file.name,
      "url": f"http://127.0.0.1:{port}/{file.name}",
      "path": str(file),
      "size_bytes": file.stat().st_size,
      "created": datetime.fromtimestamp(file.stat().st_ctime).isoformat(),
      "modified": datetime.fromtimestamp(file.stat().st_mtime).isoformat()
  }
  ```
- If `port` is None, read from `outputs/.server_status.json`
- Sort results by modified time (newest first)
- Return list of dicts

**2. format_status_report(status_dict, chart_links=None) function:**
- Format server status for human-readable output
- Input: status_dict from server_cli.get_status()
- If chart_links provided, append formatted link list
- Output format:
  ```
  服务状态: 运行中
  端口: 8100
  启动时间: 2026-05-07 14:30:00
  运行时长: 00:15:32
  
  可访问图表 (3):
  1. sales_chart.html
     URL: http://127.0.0.1:8100/sales_chart.html
     大小: 1.2 MB
     修改时间: 2026-05-07 14:45:00
  
  2. quarterly_report.html
     URL: http://127.0.0.1:8100/quarterly_report.html
     大小: 850 KB
     修改时间: 2026-05-07 13:20:00
  ```
- Return formatted string

**3. CLI interface (if __name__ == "__main__"):**
```bash
python scripts/server_status.py links [--port PORT]
python scripts/server_status.py report
```

**Error handling:**
- If html_dir doesn't exist, return empty list
- If server_status.json missing, use default port 8100
- Handle permission errors gracefully

**Dependencies:**
```python
from pathlib import Path
from datetime import datetime
import json
```

DO NOT import from server_cli.py (avoid circular dependency).
