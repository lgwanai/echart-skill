   ### 场景 12: 命令行导出

   **触发条件**：用户需要通过命令行批量导出图表或集成到自动化脚本中。

   **导出单个图表：**
   ```bash
   # 基本用法（自动生成文件名）
   python scripts/chart_cli.py export-chart chart_config.json

   # 指定输出路径
   python scripts/chart_cli.py export-chart chart_config.json --output reports/sales.html

   # 使用暗色主题
   python scripts/chart_cli.py export-chart chart_config.json --theme dark
   ```

   **导出仪表盘：**
   ```bash
   python scripts/chart_cli.py export-dashboard dashboard_config.json --output dashboard.html
   ```

   **导出甘特图：**
   ```bash
   python scripts/chart_cli.py export-gantt tasks.json --title "项目进度" --output gantt.html
   ```

   **配置文件格式：**
   ```json
   {
       "title": "销售数据图表",
       "db_path": "workspace.duckdb",
       "query": "SELECT category, SUM(amount) as total FROM sales GROUP BY category",
       "echarts_option": {
           "xAxis": {"type": "category"},
           "yAxis": {"type": "value"},
           "series": [{"type": "bar"}]
       }
   }
   ```

   **命令参数说明：**
   - `<config>`: 配置文件路径（JSON格式）
   - `--output`, `-o`: 输出HTML文件路径（可选，默认使用标题+时间戳）
   - `--theme`: 图表主题（default 或 dark，默认 default）
   - `--title`: 甘特图标题（仅 export-gantt 命令）

   **查看帮助：**
   ```bash
   python scripts/chart_cli.py --help
   python scripts/chart_cli.py export-chart --help
   ```

   **批量导出示例：**
   ```bash
   # 导出多个图表
   for config in configs/*.json; do
       python scripts/chart_cli.py export-chart "$config"
   done

   # 自动化报表生成
   python scripts/chart_cli.py export-dashboard monthly_report.json --output "reports/report_$(date +%Y%m%d).html"
   ```
   ```
