   - SQLAlchemy engine management
   - Connection pooling (engine created once, reused)

     - MySQL: `mysql+pymysql://`
     - PostgreSQL: `postgresql+psycopg2://`

- Log connection events (not credentials) via structlog

Functions to create:

1. `discover_tables(connector: SQLConnector, schema: Optional[str] = None) -> list[str]`:
   - Use `sqlalchemy.inspect(engine).get_table_names(schema)`
   - Return sorted list of table names
   - Log count and schema

2. `describe_table(connector: SQLConnector, table_name: str, schema: Optional[str] = None) -> list[dict]`:
   - Use `inspector.get_columns(table_name, schema)`
   - Return list of dicts with: name, type, nullable, default, primary_key
   - Format for display (similar to database-style table output)

3. `list_schemas(connector: SQLConnector) -> list[str]`:
   - Use `inspector.get_schema_names()`
