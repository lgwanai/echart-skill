  <action>
Create `scripts/dashboard_schema.py` with pydantic v2 models:

1. **ChartPosition model:**
   - `row: int` (ge=0, description="Row position 0-indexed")
   - `col: int` (ge=0, description="Column position 0-indexed")
   - `row_span: int` (default=1, ge=1, description="Number of rows to span")
   - `col_span: int` (default=1, ge=1, description="Number of columns to span")

2. **ChartConfig model:**
   - `id: str` (description="Unique chart identifier")
   - `position: ChartPosition`
   - `title: str` (default="", description="Chart title")
   - `query: str` (description="SQL query for chart data")
   - `echarts_option: dict` (default_factory=dict, description="ECharts option object")
   - `custom_js: str` (default="", description="Custom JavaScript code")

3. **DashboardConfig model:**
   - `title: str` (default="Dashboard", description="Dashboard title")
   - `columns: int` (default=2, ge=1, le=12, description="Number of grid columns")
   - `row_height: int` (default=400, ge=100, description="Height of each row in pixels")
   - `gap: int` (default=16, ge=0, description="Gap between charts in pixels")
   - `charts: list[ChartConfig]` (min_length=1, description="List of chart configurations")
   - `db_path: str` (default="workspace.db", description="Database path")

4. **Position overlap validator:**
   Add `@field_validator('charts')` class method `validate_no_overlapping_positions`:
   - For each chart, calculate occupied cells based on position + spans
   - Track occupied cells in a set
   - Raise ValueError if any cell is already occupied
   - Error message: "Chart {chart.id} overlaps with another chart at position {cell}"

5. **Column span validator:**
   Add `@field_validator('charts')` class method `validate_column_bounds`:
   - For each chart, check `position.col + position.col_span <= columns`
   - Raise ValueError if chart extends beyond grid columns

6. **Export `model_json_schema()` wrapper:**
   Add `def get_dashboard_json_schema() -> dict` function that returns the JSON schema.

Do NOT create dashboard_generator.py in this task - only the schema models.
Use `from pydantic import BaseModel, Field, field_validator` imports.
Follow existing code style from chart_generator.py (Chinese comments OK, ensure_ascii=False for JSON).
