# 单色旭日图

**Category:** `sunburst`
**Official:** https://echarts.apache.org/examples/zh/editor.html?c=sunburst-monochrome
**Template:** examples/sunburst-monochrome.html
**Data Format:** `[{name?: string, value?: number, itemStyle?: {}, children?: [...]}, ...]`
**Features:** per-item colors via itemStyle, emphasis/hover effects
